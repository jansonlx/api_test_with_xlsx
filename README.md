# api\_test\_with\_xlsx

* 説明  
  接口自動化測試（Python + Excel）

  * Excel：固定表格格式，維護測試用例

  * Python：讀取 Excel 表格用例，執行接口測試

* 功能（主要在 Excel 中操作）

  1. 每一行為一條接口用例，可隨時修改

  2. 可選擇執行任意接口用例

  3. 可設置收件人，接口出錯時進行郵件下發

  4. 統計每個接口執行時間（郵件下發結果）

  5. 靈活設置判斷接口是否成功的條件

**為不透露公司項目信息，把 Excel 表格裏數據修改或刪除了部分，  
此腳本無法開箱即用，僅供參考！**
